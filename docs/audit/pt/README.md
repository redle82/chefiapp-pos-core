# PT Pilot Audit Evidence

This folder holds evidence artifacts for the Portugal pilot audit.

## Structure

- evidence/legal: contracts, non-fiscal posture, SOPs
- evidence/compliance: retention sign-off, audit query outputs
- evidence/ops: backup runs, restore drill notes, incident log
- evidence/engineering: SAF-T exports, validation reports, execution logs

## Naming

Use ISO dates and short descriptions, for example:

- 2026-02-15_non-fiscal_sop_signed.pdf
- 2026-02-16_saft_export_test.xml
- 2026-02-17_backup_run_log.txt
