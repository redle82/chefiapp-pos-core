# Glossário de termos do Admin (Comando Central)

**Uso:** Termos canónicos para menus, secções e descrições do Admin. Manter consistência em toda a UI.

| Termo canónico | Evitar | Nota |
|----------------|--------|------|
| Assinatura | Subscripción, Subscription (em PT) | Página de plano e faturação |
| Plano | Plan (em PT) | Secção do menu Config |
| Configuração | Configuración, Settings (em PT) | Menu Sistema |
| Integrações | Integraciones (em PT) | Canais e integrações |
| Canais e integrações | Canais & Integrações | Secção Config |
| Operação | Operación, Operations (em PT) | Secção Config e grupo menu |
| Equipa | Team (em PT), Staff | Secção Config (Administradores, Funcionários) |
| Infraestrutura | Infrastructure (em PT) | Secção Config |
| Módulos | Modulos (sem acento) | Ativar/configurar módulos |
| Localizações | Ubicaciones, Locations (em PT) | Config identidade |
| Funcionários | Empleados, Employees (em PT) | Config equipa |
| Administradores | Admin users, Usuários admin | Config equipa |
| Software TPV | POS Software, TPV | Config operação |
| Fechos | Fechamentos (preferir Fechos em PT-PT) | Menu Finanças |
| Diretório | Directorio (clientes) | Menu Clientes |
| Catálogo | Catalogo | Menu Produto |
| Relatórios | Reportes, Reports (em PT) | Menu Inteligência |
| Dispositivos | Devices (em PT) | Menu Sistema |
| Observabilidade | Observability (em PT) | Menu Sistema |

Todos os textos visíveis do Admin devem usar estes termos na língua alvo (PT).
