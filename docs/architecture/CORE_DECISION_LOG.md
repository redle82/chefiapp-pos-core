## 2026-01-28 — CORE_SYSTEM_OVERVIEW v2 + CORE_CONTROL_PLANE_CONTRACT (correcções)

| **CORE_SYSTEM_OVERVIEW v2:** Correcções após discordância. (1) **Kernel ≠ Core:** Kernel = motor soberano de execução (gates, isolamento, eventos, retries); Core = negócio (pedidos, dinheiro); Core roda SOBRE o Kernel. Kernel = intocável absoluto; Core evolui. (2) **Hierarquia real:** Hardware → Docker → **Kernel** → **Core** → Contratos → Terminais.
