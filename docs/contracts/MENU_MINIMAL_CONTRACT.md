# Contrato Menu Mínimo (Passo 2 do Onboarding)

**Propósito:** Passo 2 do wizard. Destravar o sistema com o mínimo de menu: 1 categoria, 1 produto, preço. Pode ser feito em ~2 minutos (web ou mobile).

**Ref:** [FUNIL_VIDA_CLIENTE.md](FUNIL_VIDA_CLIENTE.md).

---

## Estado antes

- Restaurant criado (bootstrap).
- Sem menu válido para vender.

---

## Exigência mínima

- **1 categoria**
- **1 produto**
- **Preço** definido

Não é "configurar tudo"; é destravar o sistema para poder avançar para o modo de operação e primeira venda.

---

## Estado após

- **Menu:** válido (mínimo).
- Pronto para Passo 3 (escolher modo de operação).

---

## Regra

O sistema não exige mais do que isto para permitir o Passo 4 (primeira venda). O resto do menu pode ser criado depois, no Dashboard.
