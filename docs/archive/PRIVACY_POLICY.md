# POLÍTICA DE PRIVACIDADE — ChefIApp

**Versão**: 1.0.0
**Data de Vigência**: 2025-12-25
**Última Atualização**: 2025-12-25

---

## 1. INTRODUÇÃO

O ChefIApp ("nós", "nosso") respeita a sua privacidade. Esta política descreve como recolhemos, usamos e protegemos os seus dados.

## 2. DADOS RECOLHIDOS

| Tipo de Dado | Finalidade | Armazenamento |
| :--- | :--- | :--- |
| **Email** | Autenticação e comunicação | Cloud (protegido por acesso autenticado) |
| **Nome do Restaurante** | Identificação do negócio | Cloud |
| **Menu** | Exibição pública e pedidos | Cloud |
| **Pedidos** | Processamento e histórico | Local + Cloud |
| **Dados de Pagamento** | Processamento via Stripe | Stripe (nunca armazenamos cartões) |

## 3. ARMAZENAMENTO OFFLINE-FIRST

O ChefIApp utiliza arquitetura **Offline-First**:
- Dados operacionais são armazenados localmente (IndexedDB).
- Sincronização com cloud ocorre quando há conectividade.
- Você mantém controlo total sobre dados locais.

## 4. PARTILHA DE DADOS

**NÃO** partilhamos os seus dados com terceiros, exceto:
- **Stripe**: Para processamento de pagamentos.
- **Autoridades**: Quando legalmente obrigados.

## 5. SEGURANÇA

- Encriptação em trânsito (HTTPS).
- Encriptação em repouso para dados sensíveis.
- Tokens de sessão expiram automaticamente.

## 6. OS SEUS DIREITOS (RGPD)

Ao abrigo do RGPD, você tem direito a:
- **Acesso**: Solicitar cópia dos seus dados.
- **Retificação**: Corrigir dados incorretos.
- **Apagamento**: Solicitar eliminação ("direito ao esquecimento").
- **Portabilidade**: Exportar dados em formato legível.

Para exercer estes direitos, contacte: **privacidade@chefiapp.com**

## 7. COOKIES E TRACKING

Utilizamos armazenamento técnico essencial (cookies e/ou armazenamento local do navegador) para:
- Sessão de autenticação.
- Preferências de interface.

**Não** utilizamos tracking de terceiros ou publicidade.

## 8. MENORES

O ChefIApp destina-se a empresas. Não recolhemos intencionalmente dados de menores de 18 anos.

## 9. ALTERAÇÕES

Podemos atualizar esta política. A versão vigente estará sempre disponível em `/privacy`.
