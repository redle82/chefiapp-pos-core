# SALES SCRIPT — ChefIApp AppStaff

**Audience:** Donos de restaurantes (Casual Dining, Dark Kitchens)  
**Duration:** 5 minutos (elevator pitch)  
**Goal:** Compelling value prop + Beta offer

---

## A — Hook (30s)

**Setup:**
"Vocês usam LastApp? Glovo? Uber?"

*[Wait for nod]*

"Sabe qual é o problema? Vocês estão conectando ferramentas, não governando a operação."

**Hook:**
"ChefIApp não é mais uma app. É um sistema operacional para restauração."

---

## B — Problem (1 min)

**What breaks in current systems:**

1. **Jogo de whisper**: 
   - Uber diz algo → LastApp avisa → Staff não sabe → Pedido falha
   - Ninguém é responsável. Tudo falha juntos.

2. **Injustiça de turnos**:
   - João sempre pega turnos ruins
   - Maria sempre sai cedo
   - Gerente não controla nada
   - Staff resente; rotatividade sobe

3. **Conformidade invisível**:
   - HACCP? Logs perdidos.
   - Jornadas legais? Ninguém sabe se respeitam.
   - Inspeção vem? Vocês tremem.

4. **Operação + Caos**:
   - Menu não sincroniza
   - Preço está errado em 3 plataformas
   - Status travado
   - Cliente reclama; vocês perdem confiança

**Resultado:** Operação frágil, staff desmotivado, risco legal.

---

## C — Vision (1 min)

**What ChefIApp does:**

"Vocês precisam de um sistema que:

1. **Governa a verdade**:
   - Um preço. Uma realidade. Sincronizada.
   - Não plugin; é núcleo.

2. **Ordena o caos humano**:
   - Turnos justos (automaticamente)
   - Tarefas críticas rastreadas
   - Carga equilibrada por função
   - Métricas visíveis ao gerente

3. **Conformidade automática**:
   - HACCP logs imutáveis
   - Jornadas legais enforced por país
   - Documentação pronta para inspeção
   - Zero riscos invisíveis

4. **Operação resiliente**:
   - Uma verdade (não horologes a disparar)
   - Confiança: menu = preço = status
   - Feedback loop fechado (cliente → staff → gerente)"

**Vision:** Um restaurante onde tecnologia e pessoas trabalham juntos, não em conflito.

---

## F — Handling Objections

### "Já temos LastApp"

**Response:**
"LastApp é bom para conectar. Mas vocês queriam mais, né? Nós oferecemos controle total. A maioria pensa que é a mesma coisa; não é."

### "Preciso integrar com X"

**Response:**
"Integramos sensores, não donos. Glovo, Uber? Conectam em 1 semana. Nunca vai quebrar o núcleo."

### "HACCP? Legal? Pareço florista"

**Response:**
"Simples: inspeção vem, vocês têm documentação perfeita. Zeram a acta. Gostam? Renovam contrato. Nós governamos o risco invisível."

### "Quanto custa?"

**Response:**
"Beta é grátis. Pós-beta: valor por restaurante + pedidos (tipo SaaS). Mas o ROI é claro: menos rotatividade, menos risco legal, operação +40% eficiente."
