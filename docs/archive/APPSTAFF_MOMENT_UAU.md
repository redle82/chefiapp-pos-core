## 🎯 A Cena

**Cliente chama o garçom 3 vezes → 1 tarefa urgente → sem caos**

---

## 📋 Como Executar

### Setup

1. Abrir AppStaff do Garçom B (responsável pela Mesa 8)
2. Garantir que não há tarefas pendentes de chamado
3. Preparar para simular 3 chamados rápidos

### Execução

1. **Chamado 1** (0s)
   - Cliente pressiona "Chamar Garçom" na Mesa 8
   - Sistema registra evento

2. **Chamado 2** (0.5s)
   - Cliente pressiona novamente (impaciente)
   - Sistema registra segundo evento

3. **Chamado 3** (1s)
   - Cliente pressiona terceira vez (urgente)
   - Sistema registra terceiro evento

4. **Resultado** (1.5s)
   - **Apenas 1 tarefa aparece** no AppStaff do Garçom B
   - Tarefa mostra urgência aumentada (visual ou prioridade)
   - Sistema não cria spam

### Pausa para Impacto (10s)

> "Vejam: 3 chamados. 1 tarefa. Zero caos.  
> O sistema entende pressão sem criar pânico.  
> Isso é o AppStaff."
