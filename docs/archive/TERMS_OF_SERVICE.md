## 1. ACEITAÇÃO DOS TERMOS

Ao utilizar o ChefIApp ("Serviço"), você concorda com estes Termos de Serviço ("Termos"). Se não concordar, não utilize o Serviço.

## 2. DESCRIÇÃO DO SERVIÇO

O ChefIApp é um sistema de gestão para restauração que inclui:
- **TPV (Terminal Ponto de Venda)**: Sistema operacional para pedidos.
- **Setup Wizard**: Configuração inicial do estabelecimento.
- **AppStaff** (Preview): Interface de gestão de equipa (em desenvolvimento).
- **Dashboard** (Estático): Métricas ilustrativas.

> **AVISO**: Componentes marcados como "Preview" ou "Demo" utilizam dados fictícios e não persistem informações reais.

## 5. DISPONIBILIDADE E OFFLINE

O ChefIApp opera com arquitetura **Offline-First**:
- Operações são enfileiradas localmente quando offline.
- A sincronização ocorre automaticamente quando a conexão é restabelecida.
- O sistema exibe claramente o estado de sincronização ("Queued" / "Syncing" / "Applied" / "Failed").

> **NOTA**: Não garantimos disponibilidade contínua. O sistema pode estar "Em Manutenção" periodicamente.
