## 🎯 Objetivo da Apresentação

Demonstrar que o AppStaff funciona como um **sistema nervoso operacional**,
gerenciando automaticamente o trabalho sem intervenção manual.

**Mensagem central**: "O restaurante se move sozinho. As pessoas só acompanham."
