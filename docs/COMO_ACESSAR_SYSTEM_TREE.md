## 🎯 DIFERENÇA IMPORTANTE

Você está vendo a **Config Tree** (árvore de configuração do onboarding).  
A **System Tree** é uma árvore diferente que mostra o sistema operacional completo.

---

## 🔗 COMO ACESSAR A SYSTEM TREE

### Opção 1: Via URL Direta
Navegue diretamente para:
```
http://localhost:5173/system-tree
```

### Opção 2: Via Link na Sidebar do Onboarding
Agora há um link "🌳 Ver System Tree" no final da sidebar do onboarding (após o progresso geral).

### Opção 3: Via Config Tree (Após Publicação)
Após publicar o restaurante, acesse `/config` e role até o final - há um item "System Tree".

---

## 📋 O QUE VOCÊ VERÁ

### Config Tree (Onboarding) - O que você está vendo agora
- **Propósito:** Configurar o restaurante
- **Estilo:** Limpo, executivo
- **Conteúdo:** Identidade, Localização, Horários, etc.

### System Tree - O que você verá em `/system-tree`
- **Propósito:** Visualizar o sistema operacional completo
- **Estilo:** IDE-like (VS Code), escuro, técnico
- **Conteúdo:**
  - Core Engines
  - Domains
  - Installed Modules
  - Configuration
  - Operation
  - Roadmap

---

## ⚠️ IMPORTANTE

A System Tree requer que o restaurante esteja publicado (`RequireOnboarding`).  
Se você ainda não publicou, pode ver a estrutura, mas alguns dados podem não estar disponíveis.

---

## 🚀 TESTE RÁPIDO

1. Abra uma nova aba
2. Navegue para: `http://localhost:5173/system-tree`
3. Você verá a System Tree completa!
