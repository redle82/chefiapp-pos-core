## 🎯 Princípios de Framing

### ❌ Nunca Diga

- "Removemos tarefas"
- "Removemos gamificação"
- "Removemos dashboards"
- "Simplificamos o app"

### ✅ Sempre Diga

- "Tarefas viram automáticas e invisíveis"
- "Gamificação vira implícita via IQO"
- "Dashboards viram estratégicos por role"
- "Evoluímos para Sistema Nervoso Operacional"

---

## 📊 Framing por Feature

### Tarefas

**❌ Errado:** "Removemos o sistema de tarefas"  
**✅ Certo:** "Tarefas agora são automáticas e invisíveis. O sistema decide o que fazer, o funcionário apenas executa."

### Gamificação

**❌ Errado:** "Removemos XP e pontos"  
**✅ Certo:** "Gamificação agora é implícita via IQO. Medimos qualidade silenciosamente, sem distrair o funcionário."

### Dashboards

**❌ Errado:** "Removemos dashboards complexos"  
**✅ Certo:** "Dashboards agora são estratégicos e por role. Cada pessoa vê apenas o que precisa para tomar decisões."

---

## 🎯 Mensagens-Chave

1. **"Sistema Nervoso Operacional"**
   - Não é task manager
   - Não é dashboard
   - É sistema que guia

2. **"Um cérebro, múltiplos terminais"**
   - Mesma inteligência
   - Interfaces especializadas
   - Contexto adaptativo

3. **"1 ação por vez, priorizada automaticamente"**
   - Foco total
   - Zero decisões
   - Máxima eficiência

## 📝 Checklist de Comunicação

### Antes de Qualquer Comunicação Externa

- [ ] Revisar framing
- [ ] Trocar "remover" por "tornar automático/invisível/estratégico"
- [ ] Focar em benefícios, não em remoções
- [ ] Usar "Sistema Nervoso Operacional"
- [ ] Usar "1 ação por vez"
- [ ] Usar "contexto adaptativo"
- [ ] Validar com audiência
