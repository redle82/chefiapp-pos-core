# GUIA DO OPERADOR - CHEFIAPP POS CORE (PRINT)

Versao: 1.0
Data: 2026-02-08

Este guia e para imprimir e deixar ao lado do caixa/TPV.

---

## 1. Inicio do turno (abrir caixa)

1. Abrir o app no TPV.
2. Entrar com o usuario do operador.
3. Ir para /op/tpv.
4. Tocar em "Abrir Turno".
5. Confirmar valor de abertura (se houver).

Se aparecer "Pagamento pendente" ou "Core indisponivel", chamar o gerente.

---

## 2. Criar pedido (atendimento)

1. Selecionar mesa ou criar novo pedido.
2. Adicionar itens.
3. Enviar para cozinha.
4. Acompanhar status no KDS.

Dica: se o pedido nao aparece no KDS, verifique se o Core esta ativo.

---

## 3. Receber pagamento

1. Abrir o pedido.
2. Selecionar metodo (dinheiro, cartao, pix).
3. Confirmar valor total.
4. Finalizar pagamento.

Se aparecer "Caixa fechado", reabrir o turno.

---

## 4. Fechar turno (fecho de caixa)

1. Confirmar que nao ha pedidos em aberto.
2. Ir para fechamento de caixa.
3. Informar valor contado.
4. Confirmar fechamento.

Se a diferenca for maior que R$50, avisar o gerente.

---

## 5. Operacao offline

Se o sistema ficar offline:

- O banner "offline" deve aparecer.
- Continuar atendendo normalmente.
- Assim que a conexao voltar, os pedidos sincronizam.

---

## 6. Problemas comuns

1. Core indisponivel

- Verifique se o Core esta ligado.
- Chamar suporte/gerente.

2. KDS nao atualiza

- Recarregar a tela do KDS.
- Verificar se o Core esta online.

3. Pagamento nao processa

- Verificar conexao.
- Tentar novamente.
- Se persistir, usar outro metodo e avisar o gerente.
