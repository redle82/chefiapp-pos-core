🎯 UX FEATURES

  ✅ No friction
     • Max 5 passos
     • Cada passo = 1 ação principal
     • Sem formulários infinitos

  ✅ Validation
     • Feedback em tempo real (não silencioso)
     • Slug duplicate check
     • Numeric validation

  ✅ Accessibility
     • Labels + ARIA attributes
     • Keyboard navigation completa
     • Color contrast >= 4.5:1

  ✅ Mobile-friendly
     • Tested 320px+
     • Touch-friendly buttons (44px min)
     • Responsive layout

  ✅ State persistence
     • Dados acumulam sem perder ao navegar
     • Back() funciona sempre

  ✅ Skip optional
     • Step 4 (Payments) é truly optional
     • Cash-only é válido

  ✅ 1. PROMO              Desconto/oferta com urgência
  ✅ 2. HOJE TEM           Daily special (11h + 18h)
  ✅ 3. BASTIDORES         Behind-the-scenes (chef em ação)

  ✅ 8. EVENT/HAPPY HOUR   Drinks + entrada + horário

  • Reels: 2-3x/semana
  • Stories: Daily (3-5x/dia em picos)
  • Lives: 1-2x/mês
