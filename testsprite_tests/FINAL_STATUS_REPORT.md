## ⚠️ TC009: The "Good Failure"

### What Happened

**Test Code (Generated by TestSprite):**
```python
# ❌ TestSprite assumed generic action-based PATCH
lock_payload = {"action": "lock"}
lock_resp = requests.patch(
    f"{BASE_URL}/api/orders/{order_id}",
    json=lock_payload
)
```

**CORE Implementation (Correct):**
```python
# ✅ CORE implemented semantic endpoint
lock_resp = requests.post(
    f"{BASE_URL}/api/orders/{order_id}/lock",
    headers=headers
)
```

### Why This Is Good

1. **CORE never promised** to support `PATCH /api/orders/{id} { "action": "lock" }`
2. **CORE correctly implemented** `POST /api/orders/{id}/lock`
3. **System correctly rejected** invalid contract
4. **Result:** Expected, desirable, and healthy failure

**👉 If TC009 passed as-is, that would be a problem.**

### Fix Required

Update test code to use:
```python
lock_resp = requests.post(
    f"{BASE_URL}/api/orders/{order_id}/lock",
    headers=headers,
    timeout=TIMEOUT
)
```

**Expected Result:** 100% pass rate (10/10)
