2. Aplicar migration SQL usando `GUIA_APLICAR_MIGRATION_PAGAMENTOS_PARCIAIS.md`
3. Validar migration com queries SQL fornecidas

### **Fase 2: Testes (2-3 horas)**

3. Corrigir bugs críticos

2. Preparar ambiente de demo
